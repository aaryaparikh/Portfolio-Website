Description

General Working



Pre - Processing
When PhenoCam or RGB images are captured at different Time periods during the day. There can be variation in position of source of light (i.e. Sun) which can affect the calculations that are required to fit a model. Thus, Pre - Processing of images with respect to a reference is required. PhenoSoft provides the feature of White Balancing of image with respect to Selected area of image.
This White Balanced images are now used for calculating GCC/NDVI/BCC/RCC (User Defined) for each pixel of images for a selected Region Of Interest of images.
Processing
The calculated data is then used to fit a statistical model forming a curve and extracting features such as Start of season, End of Season, Length of Season, Mid point etc.
Post-Processing
After analysing the data carefully, the value for Start of Season, End of Season and Degree of Polynomial are noted and used as input for generating polynomial curve for each pixel in Region Of Interest.
Image Generation
The collection of Statistical model generated will help us to generate an image on an arbitary value provided by user. Thus, this can generate predictive image for a time on which images were not captured (Implementing the concept of moving image).

Key Features

  • Accuracy (Nothing based on assumptions, everything depends on Calculations)
  • Optimal Processing
  • Selective Processing
  • All required features in same application
