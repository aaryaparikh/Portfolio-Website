Languages Used

Description

General Working

Initially, hand movement is tracked using MPU6050 which transmits the data to Arduino board.

Arduino board processes the data and transfers the data to the system which is connected through bluetooth module HC-05.

This data is further processed and converted to instructions.

This instructions are executed using a python application running in the background.

Demonstration of Project


An I.O.T. implementation that can be used as controller for gaming, Visual Reality applications and many more.
